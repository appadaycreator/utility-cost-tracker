# 光熱費推移記録ツール【無料】簡単・無料計算 - 技術仕様書

## 概要

**サービス名**: Utility Cost Tracker
**バージョン**: 2.0.0
**更新日**: 2026-06-03
**URL**: https://appadaycreator.com/utility-cost-tracker/

電気・ガス・水道の月別コストを記録して年間推移を可視化。グラフ・目標達成度表示で節約モチベーション向上。登録不要・完全無料でご利用いただけます。

## 2.0.0 の新機能（2026-06-03実装）

### 1. グラフ表示機能
- **月別推移グラフ**：積み上げ棒グラフで電気・ガス・水道の月別推移を可視化
- **種別別構成**：ドーナツグラフで全記録における種別別割合を表示
- Chart.js を使用した動的グラフ描画

### 2. 節約目標設定・達成度表示
- 「今月の節約目標」セクションで金額を設定可能
- プログレスバーで目標達成度をリアルタイム表示
- 達成時に祝いメッセージを表示
- 目標情報もローカルストレージに保存

### 3. 誤削除防止
- 削除ボタン（×）クリック時に確認ダイアログを表示
- ユーザーが明示的に確認してから削除

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化
- **保存データ**:
  - `utility-cost-tracker`: 記録一覧（月、種別、金額）
  - `utility-cost-goal`: 目標情報（目標金額、設定日時）

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 記録したい項目を入力フォームに入力する
2. 「追加」または「記録」ボタンをクリックする
3. 記録一覧で過去のデータを確認する
4. グラフや集計で傾向を把握する
5. 継続的に記録して変化を追跡する

## よくある質問（FAQ）

**Q: 光熱費推移記録ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
